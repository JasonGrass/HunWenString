# 回文子串的查找算法

---

*问题描述：*
对于一个字符串，可以通过删除某些字符变成回文字符串，
如"cabebaf"，删除 c e f，得到 abba ，就是回文字符串。
要求，给定某一字符串，计算出最长的回文字符串。
如 abbva 有 a ,aa, bb, abba 等多个回文字符串，其中 abba 是最长的。

输入：字符串
输出：最长回文字符串的长度

e.g.
输入：abbva
输出：4
